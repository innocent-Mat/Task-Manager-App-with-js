# 📋 Task Manager App

A simple and responsive Task Manager application built with HTML, CSS, and JavaScript. This project allows users to create, edit, delete, and save tasks directly in their browser using Local Storage.

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- 📅 Assign dates to tasks
- 📝 Add task descriptions
- 💾 Automatically saves tasks using Local Storage
- ⚠️ Confirmation dialog before discarding unsaved changes
- 🔄 Tasks persist after refreshing the page
- 🧹 Automatically removes special characters from generated task IDs

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/innocent-Mat/Task-Manager-App-with-js.git
```

2. Open the project folder.

3. Launch `index.html` in your favorite web browser.

No installation or dependencies are required.

## 📖 How to Use

1. Click **Add Task**.
2. Enter a task title.
3. Choose a date (optional).
4. Add a description (optional).
5. Click **Add Task**.
6. Edit or delete tasks whenever needed.
7. All tasks are automatically saved in your browser.
